# Time Letters Privacy Policy

This repository hosts the privacy policy for the Time Letters mobile application.

## URL

The privacy policy is publicly accessible at:
**https://thegrovelabs.github.io/timeletters-privacy/**

## Setup

GitHub Pages is configured to serve the `index.html` file from the root of this repository.

## Updates

To update the privacy policy:
1. Edit `index.html`
2. Commit changes
3. Push to `main` branch
4. GitHub Pages will automatically redeploy within 1-2 minutes

## App Integration

This privacy policy is linked from the Time Letters app Settings page.
